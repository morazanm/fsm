How to Download

FSM is built in Racket, you can download Racket here.
Below you can find instructions on how to install FSM.

Raco DrRacket Graphviz
Using Raco
raco pkg install https://github.com/morazanm/fsm.git
Using DrRacket Package Manager
Instructions:
  1. File->PackageManager
  2. Under Package Source paste the following: https://github.com/morazanm/fsm.git
  3. Click Install
  4. Wait for installation
Graphviz Installation (Windows)

The graphviz library is not a required in order to use the DSL, however to use the sm-graph and visualization tool graph representation mode you must has graphviz installed and set as a path variable. A full set of instructions on how to install graphviz can be found at the graphviz website. You can also find a shortened set of instructions below.

Windows Mac Linux

Graphviz Installation (Windows)

  1. Download the appropriate stable version.
  2. Open the installer and follow the installation instructions
  3. In the bottom left of the screen type environment in the search bar
  4. Click on "Edit the system environment variables"
  5. In the Advanced tab click the Environment Variables button
  6. Click on PATH so that it is highlighted, then click Edit
  7. Then click New and add the location to where you installed graphviz
    • Make sure to highlight the /bin folder inside of the graphviz folder!
  8. Press OK
  9. C:\> dot -V

Graphviz Install (Mac)

Homebrew
brew install graphviz
MacPorts
sudo port install graphviz

Graphviz Install (Linux)

sudo apt install graphviz
