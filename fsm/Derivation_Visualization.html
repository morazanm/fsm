7 Derivation VisualizationðŸ”—

The word derivation visualization tool uses the arrow keys to move the visualization forwards or backwards. The right arrow moves the visualization one step forward. The left arrow moves the visualization one step backwards. The up arrow moves the visualization to the beginning. The down arrow moves the visualization to the end

procedure

(grammar-viz g    
  w    
  [#:derv-type derv-type    
  #:cpu-cores cpu-cores    
  inv-list]) → (or/c (void) string)
  g : grammar
  w : word
  derv-type : (or/c 'left 'right 'level-left 'level-right)
   = 'left
  cpu-cores : (or/c number? #f) = #f
  inv-list : (listof (listof state? (-> word? boolean?)))
   = (void)
Launches a visualization tool for the construction of the derivation tree for the given word using the given grammar. If the given word is not in the language then the tool does not launch and a string is returned indicating that the given word is not in the given grammar’s language. The optional keyword parameter derv-type is only used with a context-free grammar to guide the substitution of the next nonterminal: leftmost first, rightmost first, lowest level rightmost first, and lowest level leftmost first. The optional keyword argument cpu-cores is used to specify the number of cores dedicated to generating images for the visualization. The optional argument inv-list is used to provide a list of pairs, each containing a nonterminal and a predicate to determine if the word generated by the nonterminal satisfies it.
