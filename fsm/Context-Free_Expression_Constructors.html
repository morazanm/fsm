procedure

(null-cfexp) → cfexp?

Builds the context-free expression for the empty language.

procedure

(empty-cfexp) → cfexp?

Builds the context-free expression for the language that only contains the empty string.

procedure

(singleton-cfexp a) → cfexp?

  a : (or/c letter? special-char?)
Builds the context-free expression for the language that only has a single word of length 1 representing the given letter or special character: $, &, !, *.

procedure

(union-cfexp cfes ...) → cfexp?

  cfes : cfexp?
Builds a union context-free expression from the given context-free expressions.

procedure

(concat-cfexp cfes ...) → cfexp?

  cfes : cfexp?
Builds a concatenation context-free expression from the given context-free expressions.

procedure

(kleenestar-cfexp cfe) → cfexp?

  cfe : cfexp?
Builds a Kleene star context-free expression from the given context-free expression.

syntax

(make-cfe [aux-cfe ...] [result-cfe cfexp?])

 
aux-cfe = (identifier? cfexp?)
Constructs result-cfe using the provided context-free expressions aux-cfe. Order of definitions does not matter when using this constructor. Recursive references are supported when using this constructor. For example, a cfe for L = anbn is defined as follows:
(make-cfe [(EMPTY (empty-cfexp))
           (A (singleton-cfexp "a"))
           (B (singleton-cfexp "b"))
           (ANBN (concat-cfexp A ANBN B))]
          (union-cfexp ANBN EMPTY))

procedure

(cfg->cfe g) → cfexp?

  g : cfg
Returns a context-free expression for the language of the given cfg.
