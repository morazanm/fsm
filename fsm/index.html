FSM

Marco T. Morazán <morazanm@shu.edu>

 (require fsm) package: fsm

A DSL for the Automata Theory Classroom

FSM is a DSL designed to help ungraduate students understand Automata Theory by allowing them to construct and minipulate state machines and grammars. For instructions on how to download the DSL, view the patch notes, or just browse the code, please see either the offical FSM readme on Github or the new FSM website.
