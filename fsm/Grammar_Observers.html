8 Grammar Observers

procedure

(grammar-nts g) → (listof nts)

  g : grammar
Returns the nonterminals of the given grammar.

procedure

(grammar-sigma g) → alphabet

  g : grammar
Returns the alphabet of the given grammar.

procedure

(grammar-rules g) → (listof grule)

  g : grammar
Returns the rules of the given grammar.

procedure

(grammar-start g) → nts

  g : grammar
Returns the starting nonterminal of the given grammar.

procedure

(grammar-type g) → symbol

  g : grammar
Returns a symbol for the type of the given grammar: ’rg, ’cfg, or ’csg.

procedure

(grammar-derive g w) → (or Derivation string)

  g : grammar
  w : word
If the given word is in the language of the given grammar, a derivation is for it is returned. Otherwise, a string is returned indicating the word is not in the language.
