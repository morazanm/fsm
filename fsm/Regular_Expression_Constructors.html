procedure

(union-regexp r1    
  r2    
  [#:sigma sigma    
  #:pred pred    
  #:gen-cases gen-cases    
  #:in-lang in-lang    
  #:not-in-lang not-in-lang]) → regexp
  r1 : regexp
  r2 : regexp
  sigma : alphabet = '()
  pred : (-> word Boolean) = (lambda (x) #t)
  gen-cases : natnum = 10
  in-lang : (listof word) = empty
  not-in-lang : (listof word) = '()
Builds a union regular expression from the given regular expressions. The optional keyword parameters are used to automatically test the generated regular expression. The alphabet is specified by sigma. A predicate to determine if a random word generated using sigma is in the regular expression’s language is specified by pred. The number of words random words to test is specified by gen-cases. Finally, in-lang and non-in-lang specify lists of words that are and that are not in the regular expression’s language. If any of the tests fails the constructor fails and throws a recipe-based error.
