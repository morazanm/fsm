1 Constants

syntax

ARROW

The symbol used to separate the lefthand side from the righthand side of a grammar rule.

syntax

BLANK

In a Turing machine tape, this symbol denotes a blank space.

syntax

BRANCH

In a ctm description, this symbol denotes conditional branch.

syntax

EMP

The symbol denoting the empty string or character. It cannot be in the alphabet of a state machine or grammar.

syntax

DEAD

The symbol denoting the default dead state. This is the state reached when no transition rules applies to the current configuration of a state machine.

syntax

GOTO

In a ctm description, this symbol denotes an unconditional branch.

syntax

LEFT

In a Turing machine transition rule, this symbol denotes moving the head to the left.

syntax

LM

In a Turing machine tape, this symbol denotes the left end marker.

syntax

RIGHT

In a Turing machine transition rule, this symbol denotes moving the head to the right.

syntax

START

The seed symbol used to generate a new start state.

syntax

VAR

In a ctm description, this symbol denotes the introduction of a variable to abstract over the currently read symbol.
