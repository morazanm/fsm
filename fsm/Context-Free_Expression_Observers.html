procedure

  e : any?
Predicate to determine if e is a context-free expression.

(empty-cfexp? e) → Boolean

  e : any?
Predicate to determine if e is a empty context-free expression.

procedure

(singleton-cfexp? e) → Boolean

  e : any?
Predicate to determine if e is a singleton context-free expression.

procedure

(union-cfexp? e) → Boolean

  e : any?
Predicate to determine if e is a union context-free expression.

procedure

(concat-cfexp? e) → Boolean

  e : any?
Predicate to determine if e is a concat context-free expression.

procedure

(kleenestar-cfexp? e) → Boolean

  e : any?
Predicate to determine if e is a Kleene star context-free expression.

procedure

(singleton-cfexp-a cfe) → string

  cfe : singleton-cfexp?
Extracts the string in the given singleton-cfexp.

procedure

(union-cfexp-cfes cfe) → (vectorof cfexp?)

  cfe : union-cfexp?
Extracts all of the context-free expressions in the given union-cfexp.

procedure

(concat-cfexp-cfes cfe) → (vectorof cfexp?)

  cfe : concat-cfexp?
Extracts all of the context-free expressions in the given concat-cfexp.

procedure

(kleenestar-cfexp-c1 cfe) → cfexp?

  cfe : kleenestar-cfexp?
Extracts the context-free expression in the given kleenestar-cfexp.

procedure

(pick-cfexp cfe) → cfexp?

  cfe : union-cfexp?
Nondeterministically return a nested sub-cfexp from the given union-cfexp. This includes any nested union-cfexps in cfe.

procedure

(gen-cfexp-word cfe) → word

  cfe : cfexp?
(gen-cfexp-word cfe n) → word
  cfe : cfexp?
  n : natural?
Nondeterministically generate a word in the language of the given cfexp. The maximum nuber of repetitions for a Kleene star is the the optional natural number if provided. Otherwise, it is 20.
