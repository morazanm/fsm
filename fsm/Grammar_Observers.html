procedure

(grammar-derive? g w) → Boolean

  g : grammar
  w : word
Determines if the given word is in the language of the given grammar.

Note: For a given unrestricted grammar, the derivation search may not terminate.

procedure

(grammar-derive g w) → (or Derivation string)

  g : grammar
  w : word
If the given word is in the language of the given grammar, a derivation is for it is returned. Otherwise, a string is returned indicating the word is not in the language.

Note: For a given unrestricted grammar, the derivation search may not terminate.
