procedure

(check-accept? M w ...) → (void)

  M : machine
  w : word
Checks if the given machine accepts all the given words. If any word is rejected, the test fails. If M only semidecides a language, words not in the machine’s language may cause the test to not terminate.

procedure

(check-reject? M w ...) → (void)

  M : machine
  w : word
Checks if the given machine rejects all the given words. If any word is aceepted, the test fails. If M only semidecides a language, words not in the machine’s language may cause the test to not terminate.

procedure

(check-derive? G w ...) → (void)

  G : grammar
  w : word
Checks if the given grammar generates all the given words. If any word is not generated, the test fails. If w is not in the grammar’s language, the test may not terminate.

procedure

(check-not-derive? G w ...) → (void)

  G : grammar
  w : word
Checks if the given grammar does not generate all the given words. If any word is generated, the test fails. If w is not in the grammar’s language, the test may not terminate.

procedure

(check-inv-holds? pred w ...) → (void)

  pred : 
(U (-> word head-pos Boolean)
   (-> (Listof (List head-pos word)) Boolean)
   (-> word Boolean))
  w : word
Checks if the given invariant predicate holds for all the given words. If the invariant predicate does not hold for any of the words, the test fails.

procedure

Checks if the given invariant predicate fails to hold for all the given words. If the invariant predicate does hold for any of the words, the test fails.
