13 Some Useful Functions

procedure

(los->symbol l) → symbol

  l : (listof symbol)
Converts a list of symbols into a symbol by concatenating the symbols in the list from left to right.

procedure

(symbol->list s) → (listof symbol)

  s : symbol
Converts the given symbol into a list of one-character symbols.

procedure

(symbol->fsmlos s) → (listof symbol)

  s : symbol
Converts the given symbol into a list of FSM symbols symbols. For example, (symbol->fsmlos ’aS-1243b) returns ’(a S-1243 b).

procedure

(generate-symbol seed l) → symbol

  seed : symbol
  l : (listof symbol)
Generates a random symbol that starts with seed and that is not in the given list of symbols.

procedure

(symbol-upcase s) → symbol

  s : symbol
Builds a symbol that is the same as the given symbol, but with all characters in uppercase.
