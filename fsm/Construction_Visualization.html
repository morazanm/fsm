6 Construction VisualizationðŸ”—

All construction visualization tools use the arrow keys to move the visualization forwards or backwards. The right arrow moves the visualization one step forward. The left arrow moves the visualization one step backwards. The up arrow moves the visualization to the beginning. The down arrow moves the visualization to the end.

procedure

(union-viz m1 m2) → (void)

  m1 : ndfa
  m2 : ndfa
Launches a visualization tool for the construction of an ndfa that decides L = {w | w∈L(m1) ∨ w∈L(m2).

procedure

(concat-viz m1 m2) → (void)

  m1 : ndfa
  m2 : ndfa
Launches a visualization tool for the construction of an ndfa that decides L = {uv | u∈L(m1) ∧ v∈L(m2).

procedure

(complement-viz m) → (void)

  m : dfa
Launches a visualization tool for the construction of a dfa for the complement of the language decided by the given dfa.

procedure

(kleenestar-viz m1) → (void)

  m1 : ndfa
Launches a visualization tool for the construction of an ndfa that decides L = L(m1)*.

procedure

(intersection-viz m1 m2) → (void)

  m1 : ndfa
  m2 : ndfa
Launches a visualization tool for the construction of an ndfa that decides L = L(m1) ∩ L(m2).

procedure

(ndfa2dfa-viz m1) → (void)

  m1 : ndfa
Launches a visualization tool for the construction of a dfa that decides L(m1).

procedure

(ndfa2regexp-viz m1) → (void)

  m1 : ndfa
Launches a visualization tool for the construction of a regular expression for L(m1).

procedure

(regexp2ndfa-viz r) → (void)

  r : regexp
Launches a visualization tool for the construction of an ndfa to decide, L(r), the language generated by the given regular expression.
